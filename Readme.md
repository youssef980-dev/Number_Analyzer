# Number Analysis Program
## ملعومه عن البرنامج
هذا البرنامج يقوم بتحليل الرقم المدخل من قبل المستخدم ويعطيه معلومات عن كل عدد مثل إذا كان العدد زوجي أو فردي،وأولي ويعطيه قيمة جذر الرقمإذا كان عدد أولي أو لا،.
## كيفية استخدام البرنامج
1. قم بتشغيل البرنامج.
2. ادخل الرقم الذي تريد تحليله.
3. اضغط على زر "تحليل" للحصول على المعلومات عن الرقم.
