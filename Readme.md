# Number Analysis Program
## ملعومه عن البرنامج
هذا البرنامج يقوم بتحليل الأعداد المدخلة من قبل المستخدم ويعطيه معلومات عن كل عدد مثل إذا كان العدد زوجي أو فردي، إذا كان عدد أولي أو لا، وإذا كان العدد موجب أو سالب. كما يمكن للمستخدم إدخال مجموعة من الأعداد وسيقوم البرنامج بتحليل كل عدد في المجموعة.
## كيفية استخدام البرنامج
1. قم بتشغيل البرنامج.
2. ادخل الرقم الذي تريد تحليله.
3. اضغط على زر "تحليل" للحصول على المعلومات عن الرقم.
